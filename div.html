Explicação dos Posicionamentos (visível)

  • position: static — fluxo normal do documento; usado para conteúdos que seguem o fluxo sem posicionamento especial.
  • position: relative — desloca um elemento sem tirá-lo do fluxo; aqui é usado para ser referência dos elementos absolute.
  • position: absolute — posicionamento relativo ao ancestral posicionado (ex.: badges e price-tags). Bom para sobreposições controladas.
  • position: fixed — elemento fixo na viewport, independente do scroll (usado no cabeçalho).
  • position: sticky — mistura de static e fixed; o elemento "gruda" quando chega ao topo (usado no menu lateral).
  • float: left / float: right — alinham elementos lado a lado fora do fluxo normal; útil em pequenos elementos como botões de navegação.
  • display: flex — alinha e distribui itens de forma simples (usado no header e botões da hero).
  • display: grid — cria layouts bidimensionais para galerias e grade de cartões.
